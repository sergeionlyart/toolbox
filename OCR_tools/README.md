üìë Mistral OCR Tools

This repository contains easy-to-use tools for performing OCR (Optical Character Recognition) on PDF documents using the Mistral OCR API. These tools enable you to quickly extract text and images from PDF files, convert them into easily editable formats, and export the results as Markdown, HTML, JSON, or DOCX files.

üöÄ Included Tools and Functionalities:

üìå 1. CLI script: ocr_utility_pdf.py

This is a command-line interface (CLI) tool utilizing the Mistral OCR API for processing PDF documents.

Main features:
	‚Ä¢	Convert PDF to text formats (Markdown, HTML, JSON).
	‚Ä¢	Embed or extract images from PDFs.
	‚Ä¢	Output results to a file or directory.
	‚Ä¢	Simple terminal-based usage.

How to use:

python ocr_utility_pdf.py <path_to_pdf> --api-key <your_API_key> --html --output-dir <output_directory>

Example:

python ocr_utility_pdf.py "/path/to/file.pdf" \
--api-key "YOUR_API_KEY" \
--html \
--output-dir "/path/to/output/"

‚∏ª

üìå 2. Automatic PDF ‚Üí DOCX converter: ocr_pdf.py

This Python script automates the OCR processing of PDF documents, saving the results as a DOCX file with integrated images and recognized text.

Main features:
	‚Ä¢	Supports processing both local and remote PDF files.
	‚Ä¢	Automatically uploads PDFs to the Mistral OCR service.
	‚Ä¢	Saves OCR results into a DOCX document with integrated images.
	‚Ä¢	Generates a detailed JSON report containing recognized text and base64-encoded images.

How to use:

python ocr_pdf.py "<path_or_URL_to_PDF>"

Local file example:

python ocr_pdf.py "file:///path/to/file.pdf"

Results are automatically saved to:
	‚Ä¢	DOCX file: docs/processed/
	‚Ä¢	JSON file: docs/JSON_data/

‚∏ª

üìå 3. Interactive webpage for viewing and exporting OCR results: index.html

This interactive HTML page allows you to view, edit, and export OCR data from a JSON file directly in your browser.

Main features:
	‚Ä¢	Loads and displays content from a JSON file (text and images).
	‚Ä¢	Allows in-page editing of text.
	‚Ä¢	Exports results directly to a DOCX document from your browser.

How to use:
	‚Ä¢	Place the index.html file next to your JSON file (generated by the OCR scripts above).
	‚Ä¢	Specify the path to your JSON file in this line:

loadAndRender('path/to/your/file.json');

	‚Ä¢	Open the HTML file in your browser.
	‚Ä¢	Edit the text as needed, then click the ‚ÄúSave to DOCX‚Äù button.
